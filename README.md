## How to customize before publishing

1. Replace LinkedIn URLs in `index.html`.
2. Replace GitHub URLs in `index.html`.
3. Add the real GitHub URL for the AWS monitoring project.
4. Add project screenshots inside the `screenshots/` folder.
5. Replace CV files in `assets/docs/` if needed.

## Deploy with GitHub Pages

1. Create a repository called `portfolio` or `mireia-garcia-portfolio`.
2. Upload all files.
3. Go to Settings > Pages.
4. Select branch `main` and folder `/root`.
5. Save and wait for the public URL.
